Dockerfile - называем так новый image 
Docker build **** - местонахождение докерфайла 
-------переименование-------
Docker tag **** &&&:%%%
**** - айди докерфайла
&&&&&- название репозитория 
%%% - название TAG

Docker image inspect &&&&&:%%% - проверяет что находиться внутри имиджа


FROM - откуда скачиваться, какой имидж нужен

CMD - команда задаёт что делать
При неверном написании CMD команды мы не сможем выйти в оболочку баш на докер image

ENTRYPOINT это список фиксированных команд которые выполняются без изменений а CMD это команды которые мы можем изменять

LABEL- команда позволяющая привнести какие-то данные в докер, контейнер, например данные о создателе и т.д.

RUN - команда запуска(например upt-get update,upt-get install)
RUN apt-get install nginx -y 
Переменная -y указывает на то что нужно подтвердить установку.

EXXPOSE 80
EXPOSE 443/tcp
