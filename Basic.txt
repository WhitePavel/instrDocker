------------------ЗАПУСК--------------------

docker run **** - создаётся и запускается новый контейнер  в докере
docker start **** - запуск заново созданного уже контейнера, никакой новой контейнер не создаётся
docker exec -it ***** /bin/bash - позволяет зайти с помощью этой команды в баш.
docker images - команда что бы посмотреть какие image есть на сервере
------------------ПАРАМЕТРЫ--------------------

Параметр -d - запускает контейнер в фоновом режиме
Параметр —name - позволяет выдать имя контейнеру.
Параметр —rm  данном контейнере уничтожает контейнер после того как он завершается.
------------------ОСТАНОВКА--------------------

Docker system prune -a —volumes - уничтожает все контейнеры не запущенные на данный момент
Docker pause **** -ставит контейнер на паузу
Docker unpause ***** - снимает контейнер с паузы
Docker stop **** - останавливает контейнер, изящно останавливает процесс позволяя безопасно из него выйти но довольно долго
Docker kill ***** - позволяет так же остановить процесс, но немедленно и небезопасно
docker rm *** - команда удаляет данный контейнер
docker rmi *** - команда удаляет image  
**** - можно указать имя контейнера в колонке names или можно указать первые несколько символов id контейнера

------------------ИНФА--------------------

docker inspect **** - показывает всю информацию ,которая относиться к данному контейнеру.
docker stat **** - команда показывает какое кол-во ресурсов портребляет контейнер 
docker ps - позволяет посмотреть запущенные на данный момент контейнеры
docker ps -a - показывает все контейнеры которые уже были запущенные и запущенны на данный момент
docker logs ***** - показывает логи контейнера(важную инфу о нём)
docker logs -f - показывает логи в прямом режиме 
docker images - команда что бы посмотреть какие image есть на сервере
